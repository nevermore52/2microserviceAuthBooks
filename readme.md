Мой rest api петпроект , по типу to-do листа. Можно добавлять книги в библеотеку, удалять, обновлять. Так же можно добавлять авторов, если при добавлении книги указанный автор не будет обнаружен в таблице - книга не добавится.
Все данные хранятся в бд Postgresql.
Добавлен полностью независимый микросервис авторизации с jwt токенами. 
Контейнеризация docker + docker-compose
